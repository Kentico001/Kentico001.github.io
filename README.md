## Collaborate

### Getting Started
1. **Fork the repository** - Click the fork button on the top right
2. **Clone your fork**
   ```bash
git clone https://github.com/YOUR-USERNAME/Kentico001.github.io.git
cd Kentico001.github.io
```

3. **Create a new branch** for your feature or fix
   ```bash
git checkout -b feature/your-feature-name
```

### Making Changes
4. **Make your changes** to the files
5. **Commit your changes**
   ```bash
git add .
git commit -m "Description of your changes"
```

6. **Push to your fork**
   ```bash
git push origin feature/your-feature-name
```

7. **Open a Pull Request** - Go to the original repository and create a pull request from your fork

### Guidelines
- Keep commits clear and descriptive
- Test your changes before submitting
- Follow existing code style
- Be respectful and constructive

### Questions?
Feel free to open an issue or reach out. Happy contributing!
